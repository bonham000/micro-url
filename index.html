Micro-URL Generator

How this API works

1) Passing a valid URL as a parameter will return a message with your shortened URL.

2) Visiting the shortened URL will redirect you to your original page.

3) Passing an invalid URL will return invalid.

See an Example

  • Generate a valid URL like this: https:heroku-app.com/https://www.google.com
  • Sample response: Here is your micro-url: 47921
  • https://heroku-app.com/47921 now redirects to https://www.google.com
